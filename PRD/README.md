## 결정 사항

| 항목 | 결정 |
|---|---|
| 사이트 축 | 생각이 주인공 |
| 관심사 카드 | 요즘 파고드는 주제 |
| 프로젝트 표현 | 문제 → 흐름 → 결과 |
| 섹션 순서 | 생각 → 프로젝트 → 프로그램 |
| 기존 아코디언 | 채널·협업은 푸터로 흡수 |
| 글 분량 | 2~3줄, 주 1~2회 |
| 작성 방식 | 사이트에서 바로 (관리자 로그인) |
| 그라데이션 | 단어별로 색 직접 지정 |
| 모바일 | 데스크톱 전용 줄바꿈 규칙 재사용 |
| 커서 잔상 | 셰이더 색이 번지는 잔상 (데스크톱 전용) |
| 카드 등장 | 아래에서 살짝 떠오르기 |
| 성능 기준 | 노트북에서 끊김 없이 |

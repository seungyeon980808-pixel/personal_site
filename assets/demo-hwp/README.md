# HwpPalette 기능 시연 클립

인트로 영상(`hwp-preview.html` + `assets/hero-hwp.js`)의 기능 소개 3칸에 들어간다.
파일을 이 폴더에 아래 이름 그대로 넣으면 자동으로 채워진다. 넣기 전까지는
빗금 친 자리 표시판(`FEATURE 01` …)이 보인다.

| 파일명 | 기능 | 화면에 담을 것 |
|---|---|---|
| `01-palette.mp4` | 커스텀 팔레트 | 탭 전환 → 격자의 블럭을 클릭 → 한글에 즉시 반영 |
| `02-markdown.mp4` | 마크다운 변환 | 내가 정한 문법으로 적고 **Ctrl+T** → 원하는 형태로 출력 |
| `03-custom-settings.mp4` | 커스텀 설정 | 자간·장평·줄간격·글씨체·글씨크기를 프리셋 하나로 한번에 적용 |

## 녹화 규격

- **길이 5초 이상** — 화면에는 3.6초가 나간다. 앞뒤를 넉넉히 두고 찍으면 좋은 구간을
  고를 수 있다. (3.6초보다 짧으면 마지막 프레임에서 멈춘 채 남는다)
- **가로:세로 4:3** (예: 1440×1080, 1600×1200). 인트로 프레임이 4:3 이라 그대로 들어간다.
  프레임은 `object-fit: cover` 라 비율이 어긋난 만큼만 가장자리가 잘린다 —
  16:9 로 찍으면 좌우가 크게 날아가므로 녹화할 때 창 비율을 맞춰 두는 편이 낫다.
- **소리 없음** — 인트로는 무음으로 재생된다.
- 한글 창만 딱 잡아서 찍는다. 작업표시줄·다른 창이 들어가면 화면이 지저분해진다.
- 마우스 움직임은 느리고 크게. 3.6초 안에 눈이 따라와야 한다.

## 넣은 뒤

```
node tools/render-intro.js --url "http://localhost:4321/hwp-preview?clean=1" ^
                           --clips assets/demo-hwp --out out/hwp-intro.mp4
```

⚠ 주소에 `.html` 을 붙이면 `serve` 가 확장자 없는 주소로 리다이렉트하면서 `?clean=1`
을 버린다 — 제목과 조작 버튼이 영상에 그대로 찍힌다.

클립 길이가 3.6초와 크게 다르면 `assets/hero-hwp.js` 위쪽의 `CLIP` 값을 조절한다.
그 숫자 하나만 고치면 전체 타이밍이 따라 움직인다.
